# Coffee-Shop-Sales-Dashboard-Using-Excel-and-Power-BI
Analyzed Coffee shop sales data to identify trends and preferences; cleaned dataset with Excel/Power Query; applied DAX for key metrics-monthly revenue, top products, peak weekday/hours; built interactive Power BI dashboards; revealed insights—coffee &amp; tea top sellers, Mondays peak, 7–10AM busiest, post‑6PM decline.
